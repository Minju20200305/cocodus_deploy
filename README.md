## About
cocodus는 '함께, 같이'를 뜻하는 영어 단어 접두사 'co-'와 code, 그리고 us를 붙여서 지은 이름입니다.   
개발자가 되고 싶은 코딩 꿈나무, 새로운 기술에 목이 마른 개발자 모두 환영해요 🎉   
 같은 언어에 관심이 있는 모임을 찾을 수 있습니다. 여러분 가까이에 있는지도 확인할 수 있어요!
